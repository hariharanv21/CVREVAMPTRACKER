CV Revamp Bug Tracker
One-time setup · ~5 minutes
1
Create a Google Sheet
Open sheets.new — name it CV Revamp Bug Tracker.
2
Open Apps Script
In the Sheet → Extensions → Apps Script. Delete existing code. Paste the full contents of Code.gs. Save (💾).
3
Run "initialize"
Select initialize in the function dropdown → click ▶ Run. Accept permissions.
4
Deploy as Web App
Deploy → New deployment → Web app → Execute as: Me → Access: Anyone → Deploy. Copy the URL.
💡 One-time setup. After this, everyone on your team opens the same URL and sees the same bugs live from your Google Sheet.
