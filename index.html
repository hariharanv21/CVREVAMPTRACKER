CV Revamp Bug Tracker
One-time setup · 5 min
1
Create a Google Sheet
Open sheets.new. Name it CV Revamp Bug Tracker.
2
Open Apps Script
In the Sheet → Extensions → Apps Script. Delete existing code. Paste the full contents of Code.gs. Save (💾).
3
Run "initialize"
Select function initialize in the dropdown → click ▶ Run. Accept permissions.
4
Deploy as Web App
Deploy → New deployment → Web app → Execute as: Me → Access: Anyone → Deploy. Copy the URL.
💡 Do this once. Everyone on your team opens the same URL. All bugs are shared live via Google Sheets.
